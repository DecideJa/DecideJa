Como Usar o Mecanismo de Busca

Campo de Busca

Digite no campo principal o nome da universidade, área ou curso que deseja encontrar. A busca começa a funcionar automaticamente quando você digita pelo menos dois caracteres.

Botão de Buscar

Clique no ícone de lupa para disparar a pesquisa a qualquer momento. Isso atualiza a lista de resultados conforme o termo digitado.

Filtros Avançados

Use o botão de filtros para abrir opções adicionais. Escolha filtrar por universidade, área ou curso para refinar os resultados. Clique em aplicar filtros para ver os resultados atualizados.

Resultados

Os resultados aparecem abaixo do campo de busca em formato de cards. Clique em um card para ver mais detalhes da universidade ou curso correspondente.

Funcionamento Interno

O sistema usa um serviço externo para consultar o banco de dados, trazendo resultados relevantes baseados no que foi digitado e nos filtros aplicados. Termos curtos não realizam busca para otimizar desempenho.
